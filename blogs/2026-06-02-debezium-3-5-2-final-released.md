Debezium 3.5.2.Final is a maintenance release in the 3.5 stable series, upgrading the Kafka dependency to version 4.1.2 and addressing several bug fixes across connectors. Improvements include better CockroachDB default value handling and table filtering fixes. The release is available through the standard Debezium distribution channels.
