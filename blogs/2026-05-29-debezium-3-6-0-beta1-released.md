We’re excited to announce the release of Debezium 3.6.0.Beta1 , which brings a wealth of new features and improvements to the Debezium ecosystem. This release upgrades to Apache Kafka 4.3, introduces an incubating Apache Fluss sink adapter for Debezium Server, adds an incubating YashanDB source connector, and deprecates the redo_log_catalog Oracle mining strategy. On the operations side, the Debezium Operator now supports OpenTelemetry configuration and configurable image pull policies.
