Debezium 3.6.0.Beta2 is a feature-rich development release that introduces MySQL GTID tags support, a new Debezium CLI, and Spanner UUID support. The Quarkus Extensions gain column change filtering for both single and batch event capturing, plus a start and stop lifecycle API.
